# 🎮 XO Game (Tic Tac Toe)

A simple and interactive **Tic Tac Toe (XO)** game built using **HTML, CSS, and JavaScript**.

This project focuses on JavaScript logic, DOM manipulation, and building a clean interactive UI.

---

## 🚀 Features

- 🎮 Two-player mode (X vs O)
- 🧠 Smart win detection system
- 🤝 Draw detection
- 🔁 Automatic restart with 3-second countdown
- ✨ Highlight winning combination
- 📱 Responsive design (mobile + desktop)
- 🎨 Smooth hover effects and animations

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## 🎯 How to Play
- Player X starts first
- Players take turns clicking empty cells
- First player to align 3 symbols wins
- If all cells are filled → Draw
- Game restarts automatically after 3 seconds
